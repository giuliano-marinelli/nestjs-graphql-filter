## Description

#### This is an unofficial Nest package designed to generate GraphQL Filter Input Types using decorators over entities classes. Also includes TypeORM Pipe Transforms to generate Where and Order inputs for entities repositories.

Nest package designed to facilitate GraphQL development by automating the creation of GraphQL Filter Input Types through the use of decorators applied to entity classes. This package also includes TypeORM Pipe Transforms, enriching the functionality by generating Where and Order inputs for repositories associated with entities. In essence, it offers a pragmatic approach to integrating filtering capabilities seamlessly into GraphQL schemas, while concurrently enhancing querying and ordering functionalities within TypeORM-powered repositories.

## Installation

```bash
$ npm install @nestjs!/graphql-filter
```
